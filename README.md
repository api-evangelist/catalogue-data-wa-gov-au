# Western Australia Data Catalogue (catalogue-data-wa-gov-au)

Western Australia Data Catalogue is a state government open-data portal for Australia running **CKAN**, exposing a standardized catalog API over ~2,909 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/catalogue-data-wa-gov-au/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=catalogue-data-wa-gov-au-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.4

## API
- **Western Australia Data Catalogue CKAN Action API** — base `https://catalogue.data.wa.gov.au/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://catalogue.data.wa.gov.au)
- Live endpoint verified: `https://catalogue.data.wa.gov.au/api/3/action/status_show` · ~2,909 datasets
